# Attack Vector: Applications - Web and Mobile

| **Vulnerability** | **Primary Threat** | **Zero Trust Policy** | **Mitigations to be applied a smart home owner** |
| :--               | :---                      | :---:                 |  :---:                                           |
|                           |                           |                       |                                                  |
| Lack of security processes | A malicious application is available on mobile app store | Comprehensive visibility. | Verify the authenticity of the application |
|                           |                           |
| Lack of security awareness | A malware moves laterally to mobile devices and starts scanning the smart home environment | Least privilege | Where possible do not jailbreak or root your devices, especially if you do not understand the security implications |
|                           |                           |                            |
| Lack of security processes | A malicious actor brute forces themselves into the web application | Authentication before authorises | Apply MFA for all web applications that you can access from Internet. If the web application does not offer MFA be careful and make a conscious decision about purchasing the device |
|                           |                           |                            |
