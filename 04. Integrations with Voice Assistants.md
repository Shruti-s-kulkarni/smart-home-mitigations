# Attack vector: Integrations with Voice Assistants

| **Primary threat ** | **Zero Trust Policy** | **Mitigations to be applied a smart home owner** |
| :---                      | :---:                 |  :---:                                           |
|                           |                       |                                                  |
| Compromise via over sharing of information with voice assistants, such as your personal information | Sharing information on a need to know basis  | Before sharing information or chatting, please check if you need to do so with the voice assistant |
|                           |                         |
| Compromise via over sharing of information with voice assistants, such as payment cards | Authentication before authorisation | Make sure you have PIN enabled on your voice assistants for making purchases |
|                           |                         |
| Compromise via over sharing of information with voice assistants, such as emails and calendars | Sharing information on a need to know basis | Before asking any voice assistants to read your emails and/or calendar entries, please keep in mind that there may be phishing emails or malicious calendar entries which can harm smart home environments |
|                           |                              |
